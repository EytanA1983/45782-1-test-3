## 🎬 זרימת משתמש טיפוסית

```
1. משתמש נכנס לאתר
   ↓
   [Header + הודעת ברוכים הבאים + Select]

2. משתמש בוחר "צפון" מה-Select
   ↓
   [טעינה...]
   ↓
   [הצגת 4 כרטיסים: נחל תנינים, מערת הנטיפים, בניאס, כורזים]
   ↓
   [כפתור "הוסף אתר טיולים חדש" מופיע]

3. משתמש לוחץ על "הוסף אתר טיולים חדש"
   ↓
   [טופס נפתח עם כל השדות]

4. משתמש ממלא:
   - אזור: צפון
   - שם: גן לאומי חרובים
   - תיאור: יער חרובים עתיק...
   - מחיר מבוגר: 28
   - מחיר ילד: 14
   ↓
   [לוחץ "הוסף אתר"]
   ↓
   [הודעה: "אתר הטיולים נוסף בהצלחה!"]
   ↓
   [האתר החדש מופיע ברשימה]

5. משתמש רוצה למחוק אתר
   ↓
   [לוחץ על 🗑️]
   ↓
   [חלון אישור: "האם אתה בטוח?"]
   ↓
   [לוחץ "כן"]
   ↓
   [הודעה: "אתר הטיולים נמחק בהצלחה!"]
   ↓
   [האתר נעלם מהרשימה]
```

## 💡 איך לראות את האפליקציה בפועל?

### אופציה 1: Development Mode
```bash
# Terminal 1 - Backend
cd backend
npm install
npm run dev

# Terminal 2 - Frontend
cd frontend
npm install
npm run dev

# פתח דפדפן: http://localhost:5173
```

docker compose up -d
